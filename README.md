# abacus-omega

Abacus Omega is a simple University course toolbox.

![Screenshot](screenshots/main.png?raw=true)

## Installation

Use your node package manager (npm/yarn) to install dependencies.

```bash
yarn install
```

## Development

```bash
yarn start
```

## Testing

```bash
yarn test
```

## Deployment

```bash
yarn build
```
